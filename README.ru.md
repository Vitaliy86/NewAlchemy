# NewAlchemy
Воссоздание в PowerShell приложения Creative Alchemy с расширенными настройками и опциями.
Доступны французская и английская версии.

   Чем это отличается от Creative Alchemy?
   
       *  Путь реестра проверяется как в X86, так и в X86-64 варианте.
       *  Добавлена поддержка опции DisableNativeAL, позволяющей отключить вывод ALchemy на драйверы OpenAl (только CT_oal.dll) и использовать вместо них программную библиотеку Creative Software 3D Library.
                   Это позволяет картам X-fi / Audigy использовать программную библиотеку Creative Software 3D Library вместо нативных драйверов OpenAl в случае проблем с конкретной игрой.
                   Карты, выпущенные после X-fi (Recon/AE/X), могут переименовать свои драйверы sens_oal.dll в Ct_oal.dll, чтобы воспользоваться этой возможностью.
                   По личному опыту, для карт, выпущенных после X-fi, не рекомендуется полагаться на нативные драйверы openAl (переименованные sens_oal.dll в Ct_oal.dll), так как некоторые игры
                   работают с этим приёмом некорректно, например Call of Duty 1 (отсутствуют эффекты EAX).
       *  Добавлена поддержка опции отладочного журнала в графическом интерфейсе.
                   Картам, выпущенным после X-fi, потребуется переименовать sens_oal.dll в Ct_oal.dll, чтобы воспользоваться этой функцией, так как журналирование требует нативных драйверов openAl.
            
    
## Установка
На вашем ПК должен быть установлен Creative ALchemy.  
Скопируйте скрипт и папку с вашим языком (culture) в папку по вашему выбору.  
Запустите скрипт.
   
Скрипт будет использовать для перевода текста в первую очередь папку culture вашего языка, а если она отсутствует — загрузит папку en-us (например: de-DE).   
Если вы не скопируете хотя бы папку культуры en-US, текст отображаться не будет.   

## Запуск скрипта  
Создайте ярлык со следующей командой:

.\powershell.exe -WindowStyle Hidden -ep bypass -file "C:\Program Files (x86)\Creative\ALchemy\NewAlchemy.ps1"

Чтобы убрать свёрнутое окно за графическим интерфейсом, установите в свойствах ярлыка значение Run=свернуто (minimized).

## Как пользоваться
При запуске приложение NewALchemy выполнит поиск в системе поддерживаемых игр с включённым DirectSound3D. Все найденные игры будут отображены в левой панели (с заголовком «Установленные игры»). В правой панели (с заголовком «Игры с поддержкой NewALchemy») будут показаны игры, которые уже переведены на использование ALchemy. Чтобы включить поддержку ALchemy для определённой игры, выберите её в левой панели и нажмите кнопку «>>». Чтобы отменить поддержку ALchemy, выберите игру в правой панели и нажмите кнопку «<<». Вы можете выбрать сразу несколько игр и затем использовать кнопки со стрелками для их обновления. 
Скрипт создаст и будет использовать новый файл NewAlchemy.ini в папке скрипта для хранения списка игр и настроек.

## Настройки

* При запуске приложение Creative ALchemy выполнит поиск в системе поддерживаемых игр с включённым
DirectSound3D. Все найденные игры будут отображены в левой панели (с заголовком
«Установленные игры»). В правой панели (с заголовком «Игры с поддержкой ALchemy») будут показаны
игры, которые уже переведены на использование ALchemy.
Чтобы включить поддержку ALchemy для определённой игры, выберите её в левой панели и нажмите
кнопку «>>». Чтобы отменить поддержку ALchemy, выберите игру в правой панели и
нажмите кнопку «<<». Вы можете выбрать сразу несколько игр и затем использовать
кнопки со стрелками для их обновления. 

* «Буферы» используется для задания количества аудиобуферов, используемых внутренне. Значение по умолчанию — 4,
подходит для большинства приложений.

* «Длительность» используется для задания длины в миллисекундах каждого из аудиобуферов.
Значение по умолчанию — 25 мс.
Общая длительность внутренней аудиоочереди равна Буферы * Длительность (то есть
100 мс по умолчанию). Может потребоваться поэкспериментировать со значением Длительности, чтобы
найти оптимальный баланс между производительностью и качеством для каждой игры. Кроме того, некоторым играм
требуются меньшие значения, чем 25 мс по умолчанию, так как они используют очень маленькие буферы DirectSound
для потоковой передачи, либо им требуется более быстрое обновление позиции воспроизведения. Уменьшение
значения «Длительность» может предотвратить сбои звука, щелчки и потрескивания. Однако более низкие значения
означают повышенную вероятность прерывания звука в моменты интенсивной нагрузки на процессор
(например, при активном обращении к диску во время загрузки уровня). Рекомендуемый подход — попробовать
настройки по умолчанию, и если звуковые артефакты слышны регулярно, попробовать снизить Длительность на
5 мс и повторить попытку. Если проблема сохраняется, попробуйте уменьшить значение ещё на 5 мс
и так далее (минимально допустимое значение — 5 мс).

* «Максимальное количество голосов» используется для задания максимального числа аппаратных голосов,
которые будет использовать ALchemy. Используемое количество голосов будет равно меньшему из значений: лимита аппаратных
голосов и данной настройки. Значение по умолчанию — 128, это максимальное число
голосов, доступное на картах SB X-Fi. Уменьшая это значение, можно зарезервировать аппаратные
голоса для другого приложения либо повысить производительность за счёт потоковой передачи меньшего числа
аудиоканалов.

* «Отключить Direct Music» используется для отключения поддержки DirectMusic. По умолчанию значение — false
(флажок снят), то есть поддержка DirectMusic включена. На данный момент не известно о проблемах,
вызванных сочетанием ALchemy с играми (например, TRON 2.0), использующими
DirectMusic. 

* «Отключить нативный AL» используется для того, чтобы заставить обёртку Alchemy игнорировать драйверы OpenAl (только Ct_Oal.dll)
и использовать собственную библиотеку (Creative Software 3D Library).
По умолчанию эта настройка имеет значение False для всех карт X-FI / Audigy.
Карты, выпущенные после X-fi, не имеют драйверов CT_oal.dll, поэтому по умолчанию эта настройка не оказывает эффекта.

Все опции журналирования требуют для работы драйверов CT_oal.dll, журналирование не работает с программной библиотекой Creative Software 3D Library.
Карты, выпущенные после X-fi, могут переименовать свои драйверы Sens_oal.dll в Ct_oal.dll, чтобы получить возможность журналирования.
Включение журналирования может повлиять на производительность звука.

* LogDirectSound: журналирование DirectSound (по умолчанию False) в файл dsound.txt
* LogDirectSound2D: журналирование DirectSound 2D в файл dsound.txt (по умолчанию False).
* LogDirectSound2DStreaming: журналирование потоковой передачи DirectSound 2D в файл dsound.txt (по умолчанию False).
* LogDirectSound3D: журналирование DirectSound 3D в файл dsound.txt (по умолчанию False).
* LogDirectSoundListener: журналирование DirectSound Listener в файл dsound.txt (по умолчанию False).
* LogDirectSoundEAX: журналирование EAX в файл dsound.txt (по умолчанию False).
* LogDirectSoundTimingInfo: журналирование информации о таймингах DirectSound в файл dsound.txt (по умолчанию False).
* LogStarvation: журналирование нехватки буферов (starvation) в файл dsound.txt (по умолчанию False).
